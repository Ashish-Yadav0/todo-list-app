Smart To-Do List Web App
This is a simple and interactive To-Do List Web Application built using HTML, CSS, and JavaScript. It helps users manage daily tasks efficiently by allowing them to add, complete, and delete tasks.
The app uses browser localStorage to store tasks, so data remains saved even after refreshing or reopening the page.
🚀 Features
➕ Add new tasks
✅ Mark tasks as completed
❌ Delete tasks
💾 Data stored using localStorage
📱 Simple and responsive UI
🛠️ Tech Stack
HTML
CSS
JavaScript
